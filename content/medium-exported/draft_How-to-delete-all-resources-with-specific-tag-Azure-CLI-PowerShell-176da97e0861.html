How to delete all resources with specific tag Azure CLI PowerShell



az group list --query "[?tags.automationToolkit == 'true' && tags.example == 'true'].name" -o tsv | ForEach-Object { Write-Host "Removing resource $_"; az group delete --no-wait --yes -n $_ }
