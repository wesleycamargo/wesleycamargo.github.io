DataOps on Azure - Authorizing an Azure Data Factory to manage a Databricks with Managed Identities and Az DevOps

In big data architectures, it’s a common need to have an orchestrator managing other resources. On Azure, this tool is the Azure Data Factory and we can handle many resources like SQL Servers, Storage Accounts, Databricks, among others. A prerequisite for this orchestration is that ADF has permissions on the resource that it must control, and this is the motivation for using Managed Identities: granting these permissions simply and quickly.
