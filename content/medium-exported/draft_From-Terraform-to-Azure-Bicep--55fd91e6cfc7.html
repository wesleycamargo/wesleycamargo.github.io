Linkedin feedback

- Bicep is essentially an ARM transpiler to a nicer (more HCL esque) language, therefore has all the advantages and disadvantages of ARM.
- Terraform keeps state where Bicep doesn’t, this has varied consequences.
- Bicep will gain newer features in Azure faster than Terraform due to it being so closely related to ARM which always gets new features first.
- Terraform will allow you to use other providers and hook into other systems where Bicep will make this difficult.
- Terraform can do destroy and plan where this functionality is either missing or poorly implemented in ARM and thus Bicep.
